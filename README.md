# 📊 Retail Store Sales Analysis (Power BI Dashboard)

## 📌 Overview
This project analyzes retail sales data from **FashionWave**, a multi-store fashion retailer.  
The goal is to extract insights from transactional data and build an interactive **Power BI dashboard** that helps decision-makers understand store performance, product demand, and profitability.

The dashboard highlights key retail metrics such as sales trends, profit margins, discount impact, and product category performance.

---

## 🎯 Business Objectives
- Improve **weekend sales performance**
- Identify **high-performing and underperforming stores**
- Analyze **product category demand**
- Understand the **impact of discounts on profit margins**
- Optimize pricing and promotional strategies

---

## 📊 Dataset Information

Dataset: `fashionwave_sales.csv`

Key columns include:

| Column | Description |
|------|-------------|
| InvoiceNo | Unique transaction ID |
| Date | Transaction date |
| StoreID | Store identifier |
| ProductID | Product identifier |
| Category | Product category |
| Quantity | Units purchased |
| UnitPrice | Price per unit |
| Discount | Discount applied |
| CostPerUnit | Cost of product |
| CustomerType | New / Returning |
| PaymentMode | Payment method |
